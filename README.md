# PPBMidtermProject

**Nama aplikasi:** Touch Up Image Editor

**Deskripsi:** Aplikasi untuk mengedit image yang memiliki fitur rotate, crop, dan basic postprocessing

**Implementasi:** CRUD, Firebase Auth (untuk akun user), Firebase Storage (menyimpan data image), Notifications (upload image), Camera, Image Picker, Image manipulation (rotate, crop, basic post-processing)
